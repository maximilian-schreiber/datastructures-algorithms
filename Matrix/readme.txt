MATRIX - Datenstruktur

Aufgabenbeschreibung:
---------------------
Implementieren Sie einen abstrakten Datentypen (Klasse Matrix), um Standardoperationen auf 
ganzzahligen Matrizen anwenden zu können.

Beschreibung der Klasse Matrix
------------------------------
Attribute:
- int[][] daten; ... 2-dimensionaler int-Array zur Speicherung der Matrizenkoeffizienten
  Die erste Dimension gibt die Reihe, die zweite Dimension die Spalte der Matrix an
  Beispiel: daten[0][1] behinhaltet den Wert der ersten Reihe und zweiten Spalte 
- int numRows, numCols; ... Anzahl der Zeilen und Spalten 

Operationen:
- Matrizenaddition: add(Matrix)
- skalare Multiplikation: mult(int)
- Matrizenmultiplikation: mult(Matrix)
